#  GET Facebook Friends

This is a project that uses Selenium to scrape friends' information from Facebook and add it to a Neo4j database.

## Table of Contents
- [Requirements](#requirements)
- [Installation](#installation)

## Requirements
- Python 3.x
- Neo4j
- WebDriver corresponding to the browser you are using (Chrome, Firefox, etc.)
